### 功能特性

- [x] 支持Emby
- [x] 支持播放视频
- [x] 支持苹果芯片的Mac
- [x] iOS内置VLC播放器
- [ ] Android使用MPV播放器
- [ ] 支持订阅视频更新
- [ ] 完善用户界面
- [ ] 完善播放器手势、字幕和配置支持
- [ ] 支持Windows
